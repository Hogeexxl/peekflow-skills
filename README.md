# PeekFlow Skills

Remote skill catalog and distributable skills for PeekFlow.

## Catalog URL

Use this URL in PeekFlow:

`https://raw.githubusercontent.com/Hogeexxl/peekflow-skills/main/skills_catalog.json`

## Layout

- `skills_catalog.json`: remote catalog entrypoint consumed by PeekFlow.
- `skills/`: distributable skill packages.
